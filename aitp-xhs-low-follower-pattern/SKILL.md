# 低粉爆款拆解：拆能复制的，不抄不能复制的

## 核心原则

对新手来说，大号爆款混了粉丝基础、品牌认知、历史权重，不一定能复制。更有价值的样本是**粉丝不多但近期数据突出**的内容。

本 skill 把"我喜欢这篇"变成结构分析：关键词入口、发布时间、账号粉丝量、封面标题点击承诺、开头三句、正文组织、停留理由、收藏理由、评论区反馈、标签组合。

爆款判断公式：爆款不是绝对数据高，是在小流量测试里明显优于平均。拆解须把样本拆三段——点击率靠封面标题，停留时长靠开头和结构，互动率靠收藏 / 评论 / 转发背后的价值或共鸣。

拆解目标不是复制原文，是提取可迁移骨架，再换成你自己的经历、观点、素材。

对标要做"五重过滤"：样本是否接近商业目标、机制能否看懂、结构能否用自己素材复用、是否剔除"我不一样"的主观噪音、是否避免空谈赛道。共鸣诊断还要拆出内容为什么让人想说、想用、想站队、想分享给第一批人。

视觉也要拆：低粉高数据样本不只拆标题正文，还要拆封面构图、3:4 画幅、信息层级、主视觉、页间节奏、是否有统一视觉母版、是否避免 PPT 感和廉价模板感。

## 什么时候用

1. 已定赛道，但不知道什么内容能被平台接受。
2. 想找对标账号却只会收藏大号。
3. 有 5-10 篇样本，希望 Agent 总结规律。
4. 要把一个爆款结构改写成自己的内容。
5. 某条数据异常好 / 差，想知道卡在点击 / 停留 / 互动哪。

## 执行步骤

1. **定义搜索词**：从赛道提 5-10 个核心词 / 问题词 / 人群词。每词能直接进小红书搜索框。

2. **筛低粉高数据样本**：记账号粉丝量、发布时间、赞藏评、标题、封面承诺。至少得 5 个"粉丝不多但数据明显高于同类"的样本。

3. **五重对标过滤**：判样本是否接近变现、机制能否懂、能否用自己素材复用、是否排除主观偏见、是否没陷抽象赛道讨论。只留值得学的，不只热闹的。

4. **拆内容骨架**：每篇拆封面承诺、标题句式、开头三句、正文段落功能、结尾动作、标签。输出样本对比表。

5. **拆三指标杠杆**：判样本强在点击率 / 停留 / 互动哪，记对应证据。每篇有"入口强点 / 内容强点 / 互动强点"判断。

6. **识别五类爆款需求**：判内容满足情感共鸣 / 见证成长 / 获得新知 / 学习提升 / 身份表达哪类或几类。不只拆形式，也拆用户为什么点 / 看 / 藏 / 评 / 转。

7. **共鸣原因解码**：判样本是否释放用户不敢说的话、满足具体使用动机、提供立场框架、让第一批人愿意转述、强化某种信念。输出"它为什么被互动"的心理原因，不只排版结构。

8. **总结可迁移公式**：提炼 3-5 个可复用标题模板 + 1 个正文结构模板。模板不含原作者专属事实，可替换成用户素材。

9. **拆视觉骨架**：记封面标题占比、主视觉位置、背景风格、字体层级、强调色、页码角标、内页结构、页面节奏。输出"可复用视觉公式"——封面冲击、内页卡片、对比页、案例页、总结页如何组织。

10. **判视觉是否值得学**：查样本手机端可读否、有收藏价值否、像小红书图文而非 PPT 否、有统一风格否。只学视觉结构，不复制原作者图片 / 素材 / 专属设计元素。

## 边界

- 用户还没定服务人群或变现路径 → 先调定位 skill。
- 用户要求逐字模仿某篇 → 提醒只提取结构和逻辑。

## 失败模式

- 只看赞，不看粉丝量和发布时间。
- 只看终数据，不拆点击 / 停留 / 互动分别在哪里。
- 把原文表达搬进自己笔记，账号缺真实差异。
- 样本太少就总结公式。
- 看一篇爆了就误以为找到公式，没连续验证。
- 对标对象与商业目标无关，只数据好看。
- 只拆标题句式，不拆它让谁获得表达 / 使用 / 站队理由。
- 只学封面颜色和装饰，不拆信息层级 / 页间节奏 / 手机端可读性。
- 把别人视觉样式整体照搬，账号缺自己的视觉识别。

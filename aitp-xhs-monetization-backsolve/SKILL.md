# 变现倒推：先想清楚靠什么赚钱，再决定发什么

## 核心原则

做 AI 工具类账号，新手最容易踩的坑是上来就问"我该发什么"。正确顺序是反过来的：先确定这个号**最终靠什么拿到商业回报**，再倒推内容应该吸引谁、解决什么问题、建立什么信任、避开什么表达。

原因很简单——变现方式决定了账号的一切：
- 接商单（工具厂商投放）：要证明你有影响力和转化力
- 卖咨询 / 陪跑：要证明你的专业判断能解决别人的具体问题
- 带货 / 课程：要证明使用场景真实、购买理由充分
- 私域 / 社群：要证明你能提供长期关系价值

另一层真相：定位不是坐在那里脑暴出来的，而是在**时机、个人优势、持续输出能力、早期内容验证**里跑出来的。AI 工具赛道有两条主路——追热点做爆款（快但短），或经营个人 IP 做长期信任名片（慢但稳）。选哪条要服务于你的真实目标，而不是哪个听起来更香。

所以定位=「变现路径 + 目标人群 + 内容承诺 + 人设证据 + 时机判断 + 长期表达力 + 禁区边界」的组合，不是一句漂亮简介。

## 什么时候用

1. 想做小红书，但只说出"我该发什么"。
2. 有几个赛道候选（AI 写作 / AI 绘画 / 办公自动化 / 效率工具测评），不知道哪个更好变现。
3. 发了一阵子，发现粉丝和你的产品对不上。
4. 想把主业 / 副业 / 咨询 / 课程做成账号。
5. 纠结追热点还是做个人 IP。
6. 兴趣很多，想判断哪个才是自己不可替代的优势。

## 执行步骤

1. **盘点可售卖资产**
   问清用户已有的产品、技能、资源、服务、社群、渠道、可投入时间。至少列出 3 个市场可能买单或交换的资产。

2. **明确 offer 与信任路径**
   写清：卖什么（产品 / 服务 / 资源包 / 咨询 / 课程 / 社群）、用户为什么需要、凭什么信你、内容如何自然接出下一步关系。不能只说"以后变现"，必须有一个可验证的付款理由或信任资产。

3. **选主变现路径**
   在商单 / 带货 / 店铺 / 咨询 / 课程 / 社群 / 私域里选 1 主 1 副。能说清"谁为什么会为我付费"。

4. **判断时机与长期表达力**
   评估当前平台风口（如国产 AI 工具爆发期）、用户需求变化、已有积累、能否连续输出 3-6 个月。能说清"为什么现在做、我凭什么长期讲下去"。

5. **选账号打法**
   在"追热点做爆款"和"个人 IP 长期信任"间定主路径；必要时设计主号 / 小号矩阵。用户清楚自己追求快流量、长信任还是分阶段组合。

6. **反推定位**
   产出一句话定位、目标用户、内容承诺、人设证据、内容边界。定位要能回答"我帮谁，在什么场景下，解决什么问题，为什么是我"。

7. **生成内容方向矩阵**
   按信任建立 / 问题解决 / 案例展示 / 资源工具 / 个人经历拆 5 类，每类至少 3 个可发布方向。

8. **定视觉定位假设**
   按变现路径选主风格 + 辅助风格（如专业系统卡、深色科技杂志、数据报告风、工具实测截图风）。定位同时回答"我说什么"和"我看起来像什么"。

9. **设计 10-20 条验证计划**
   把定位拆成 2-4 个内容方向，用前 10-20 条验证点击 / 停留 / 收藏 / 评论 / 关注 / 私信信号。定位是可测试计划，不是一次性结论。

## 边界

- 用户已给明确定位、只要写某篇笔记 → 用选题 / 文案类 skill。
- 用户只问平台规则、图片尺寸、发布时间 → 不在此 skill 范围。

## 失败模式

- 把"我喜欢什么"当唯一定位，忽略商业路径。
- 为短期流量选自己无法长期讲的赛道。
- 一开始设计太多变现方式，内容承诺模糊。
- 过早把定位写死，不用前 10-20 条验证。
- 只追风口没个人积累，流量来了接不住。
- 只问"发什么"，没定义产品 / 服务 / 信任资产 / 关系下一步。
- 只定内容赛道不定视觉信任感，封面内页长期不统一。
